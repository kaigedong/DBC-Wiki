# DBC-Blockchain 主网 RPC

发送 RPC 请求的方式，请参考上一篇文章。

并不是所有的链上数据都有 RPC 接口。如果想要查询的数据（包括历史数据）没有通过 RPC 暴露出来，可以通过查询链上存储来获取对应数据。

# 1. onlineProfile 模块

onlineProfile 模块记录了机器在线奖励的信息
